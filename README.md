# ⚡ 포켓몬 도감 앱

## 👩‍💻 연습

포켓몬 도감 앱은 다양한 포켓몬을 내 포켓볼에 선택할 수 있고 세부 정보를 확인할 수 있습니다. 각 포켓몬의 이름, 이미지, 속성, 설명을 제공하며, 사용자 경험을 향상시키기 위해 Redux와 Context API를 사용해 상태 관리를 최적화했습니다
